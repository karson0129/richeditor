# richeditor
一个利用重写webview实现的富文本

使用方法

#Step1

Add it in your root build.gradle at the end of repositories:

allprojects {

		repositories {
		
			...
			maven { url 'https://jitpack.io' }
			
		}
		
	}
  
  #Step2
 
  dependencies {
  
	        implementation 'com.github.karson0129:richeditor:Tag'
		
	}
